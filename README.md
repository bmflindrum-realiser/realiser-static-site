# realiser.ai SEO-pakke

Filer å kopiere til repo-rot for GitHub Pages:

- `index.html` – minimal landingsside med SEO-optimalisert <head> og CTA
- `robots.txt`
- `sitemap.xml`
- `CNAME` – må inneholde `realiser.ai`

Legg i tillegg til disse filene i rot når du har dem klare:
- `/og-cover.jpg` (1200x630)
- `/favicon.ico`
- `/apple-touch-icon.png` (180x180)
- `/logo.png`
- `/hero.webp` eller `/hero.jpg`

Sjekkpunkter i GitHub Pages:
- Aktiver HTTPS i repo-innstillinger.
- Verifiser at `https://realiser.ai/robots.txt` og `https://realiser.ai/sitemap.xml` fungerer etter deploy.
